## 📝 撮影のコツ

### 照明
- 自然光または白色LED照明を使用
- 影を避けるため複数方向から照明

### 背景
- 白色または グレーの無地背景
- 余計な物が写り込まないよう注意

### アングル
- 水平・垂直を意識した構図
- 特徴的な部分がよく見えるアングル

### 解像度
- 最低1920×1080以上
- Web表示用は適宜リサイズ

## ファイル形式・命名
- 形式: JPG（写真）、PNG（レンダリング・UI）
- 命名: `説明_詳細.拡張子`
- 例: `main-assembly_angle1.jpg`
